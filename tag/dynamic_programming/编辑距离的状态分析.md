### 基本描述
- 删除一个字符 deletion
- 插入一个字符 insertion
- 替换一个字符 substitution

这里看上去很清楚, 但其实很容易就糊涂起来, 因为没有主语.

- 删除`A`末尾一个字符 deletion
- 用`B`末尾插入一个字符 insertion
- 把 `A` 末尾字符替换成`B`末尾的一个字符 substitution


---
当 A[i]!=B[j]
### 从 D(i-1,j-1)到 D(i,j)
当前:
```
A[0,i-1] = B[0,j-1]
```
对于 D(i,j),直接将A[i]替换为B[j]
